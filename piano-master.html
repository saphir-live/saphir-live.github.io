CONFIGURATION

1. Remplacez firebaseConfig dans le <script type="module"> avec vos clés Firebase (Realtime Database activé).

2. Branchez votre piano Yamaha en USB et autorisez l'accès MIDI dans le navigateur quand demandé.

3. Ouvrez slave.html sur l'écran d'affichage (peut être un autre appareil sur le même réseau).

Navigateurs supportés : Chrome / Edge (Web MIDI API requis).
