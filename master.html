⚙ FIREBASE CONFIG

Pour utiliser cette application, remplacez la configuration Firebase dans la section <script type="module"> en haut du fichier avec vos propres clés.

Créez un projet sur console.firebase.google.com, activez le Realtime Database, puis copiez votre configuration dans les deux fichiers master.html et slave.html.

Réglez les règles de sécurité sur read: true, write: true pour les tests.
